# Adidas US Sales-Data-analysis(Interactive Dashboard creation using Microsoft Power BI)
## Project Objective
This project presents an end-to-end Power BI analysis of Adidas US Sales, covering sales performance, profitability, product trends, customer behavior, and key business insights. The dashboard helps business stakeholders understand sales distribution across time, regions, states, retailers, and product categories.
## Dataset used
- <a href="https://github.com/bharathi2494/Power-BI--2025--Adidas-US-sales-Dashboard/blob/main/Adidas%20US%20Sales%20Datasets.xlsx">Dataset</a>
## The Adidas Sales Analysis dashboard provides:
- Total Sales Performance
- Operating Profit
- Units Sold
- Average Price per Unit
- Operating Margin
- Sales Trends (Monthly, Regional, State-wise)
- Product Category Insights
- Retailer Performance
- Key Influencers Analysis
